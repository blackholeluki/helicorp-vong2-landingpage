# HELICORP - DỰ ÁN PHÁT TRIỂN WEBSITE (VÒNG 2)

Dự án Landing Page giới thiệu siêu phẩm card đồ họa **ROG Astral RTX 5090 Edition 20**, được xây dựng trong khuôn khổ bài kiểm tra năng lực Vòng 2 của HELICORP.

**Link Deploy thực tế:** [Chèn link Vercel của bạn vào đây]

---

## Công nghệ sử dụng
Dự án được xây dựng theo hướng tối ưu hóa hiệu năng, hạn chế phụ thuộc vào các framework đồ sộ:
- **Core:** HTML5, CSS3, JavaScript (Vanilla).
- **Styling:** Tailwind CSS (CDN) kết hợp Custom CSS (Glassmorphism, Animations).
- **Libraries:** AOS (Animate On Scroll) cho hiệu ứng cuộn trang, Remix Icons.

---

## Tính năng nổi bật & Điểm cộng đã hoàn thiện
Bám sát yêu cầu từ đề bài, dự án đã hoàn thiện cấu trúc tiêu chuẩn và tích hợp nhiều "điểm cộng" nâng cao:

1. **Tư duy UI/UX & Responsive:** 
   - Thiết kế mang âm hưởng Cyberpunk, tông màu đỏ/đen/bạc đặc trưng của ROG.
   - Hiển thị mượt mà trên cả Desktop, Tablet và Mobile.
2. **Tối ưu Hiệu năng & SEO:** 
   - Hình ảnh được nén định dạng WebP, tự lưu trữ (self-hosting).
   - Tích hợp đầy đủ các thẻ Meta cơ bản (Title, Description, Open Graph).
3. **Phát triển chế độ Dark Mode:** 
   - Chuyển đổi giao diện Sáng/Tối mượt mà thông qua JavaScript và Tailwind CSS.
4. **Scrollytelling & Parallax:** 
   - Section "Tản nhiệt" được thiết kế dạng kể chuyện cuộn trang (Sticky Scroll) kết hợp hiệu ứng thay đổi đồ họa trực quan.
5. **AI Chatbot Tư vấn (Gemini API):** 
   - Tích hợp cửa sổ Chatbot góc màn hình.
   - Kết nối trực tiếp với API của Google Gemini để tự động tư vấn cấu hình và thông tin sản phẩm.
6. **Micro-interactions:** 
   - Trải nghiệm Form đăng ký với hiệu ứng Focus, Gradient Hover, và giả lập độ trễ khi xử lý dữ liệu.
